# Turbojet Preliminary Design Tool (TPDT)

A Python-based interactive command-line tool for preliminary turbojet aircraft sizing.

## Capabilities
- Estimates takeoff gross weight
- Calculates required thrust
- Computes wing loading and geometry
- Determines stall velocity

## Purpose
This project translates preliminary aircraft performance formulas into a computational model for early-stage design evaluation.

## How to Run
```bash
python "Turbojet Preliminary Design Tool.py"
